# Cursor Snake Test

Minimal, classic Nokia-style Snake game built with plain HTML/CSS/JS.

## Run locally

1. Open `index.html` in a browser.
2. Use arrow keys to move.
3. Press `R` or click Restart to start over.

## Controls

- Arrow keys: move the snake
- R: restart
- Grid size input (10â€“20): change grid size, then restart

## Notes

- Game ends on wall or self collision.
- Score counts foods eaten.
